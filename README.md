# 🚀 AlgoPath

### AI-Powered Personalized Learning Platform

AlgoPath is an AI-powered learning platform designed to help students master **Data Structures & Algorithms**, **Web Development**, and **Data Analysis** through personalized learning paths, interactive visualizations, coding practice, and intelligent progress tracking.

---

## ✨ Features

* 📚 Personalized learning roadmaps
* 🧠 AI-powered study planning
* 💻 Interactive coding practice
* 📈 Progress tracking dashboard
* 🎯 Structured DSA learning path
* 🌐 Web Development learning path
* 📊 Data Analysis learning path
* 🔒 Secure user authentication
* ⚡ Fast and responsive user interface
* ☁️ Modern full-stack architecture

## ⚡ Quick Start

### Prerequisites

Make sure the following software is installed:

* Python 3.11+
* Node.js 20+
* npm
* Git
* MongoDB (Local or Atlas)

---

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/tulasidhar-p/algopath.git
```

```bash
cd algopath
```

---

### 2️⃣ Backend Setup

Navigate to the backend folder:

```bash
cd backend
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Start the backend server:

```bash
uvicorn app.main:app --reload
```

---

### 3️⃣ Frontend Setup

Open another terminal.

Navigate to the frontend directory:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the React development server:

```bash
npm run dev
```

---

### 🚀 Run the Application

Frontend:

```text
http://localhost:5173
```

Backend API:

```text
http://127.0.0.1:8000
```

API Documentation:

```text
http://127.0.0.1:8000/docs
```
