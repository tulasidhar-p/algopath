# 🚀 AlgoPath

### AI-Powered Personalized Learning Platform

AlgoPath is an AI-powered learning platform designed to help students master **Data Structures & Algorithms**, **Web Development**, and **Data Analysis** through personalized learning paths, interactive visualizations, coding practice, and intelligent progress tracking.

---

## ✨ Features

* 📚 Personalized learning roadmaps
* 🧠 AI-powered study planning
* 💻 Interactive coding practice
* 📈 Progress tracking dashboard
* 🎯 Structured DSA learning path
* 🌐 Web Development roadmap
* 📊 Data Analysis roadmap
* 🔒 Secure authentication system
* ⚡ Modern responsive UI
* ☁️ FastAPI + React architecture
